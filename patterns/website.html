Notes

Decisions carried from primitives.

Icon direction

Lucide icon library. Stroke weight 1.75. Sizes: 14px inside sm buttons, 16px inside md buttons, 18px inside lg buttons. Plain 20px for nav or body use. 24px for feature treatments. Inherit currentColor. No fills. No custom icons unless the brand needs one.

Type in marketing register

Fraunces italic weight 500 reserved for wordmark. Fraunces roman weight 500 for hero and section headlines. Inter 400 body, Inter 500 for UI labels and buttons. JetBrains Mono for case results and metric captions. Sizes match this page.

Color discipline

Warm-800 is page default. Warm-900 for hero and metric row contrast. Warm-950 for CTA band and input fills. Signal only on: logo tile, single hero CTA, eyebrow labels for emphasis, case result lines. Zero Feature cards on public marketing surfaces. Primary button is terracotta.

Asset exports needed for Phase 4

  • Logo tile SVG master + PNG 32 / 64 / 128 / 256 / 512 / 1024
  • Favicon: 16 / 32 / 180 (apple-touch) / SVG
  • OG / Twitter card: 1200×630 with tile + wordmark on Warm-900
  • LinkedIn company cover: 1128×191 tile at left, tagline right
  • Wordmark SVG (italic-500-opsz144) for headers and email signatures
  • Tile + wordmark horizontal lockup SVG (nav use)
